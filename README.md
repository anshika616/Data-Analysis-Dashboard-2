# Amazon Sales Data Analysis (Interactive dashboard using powerbi)

## Project Objective

To analyze and monitor Amazon sales performance by category, product, and order status, 
while identifying top-performing cities and states to support data-driven business decisions.

## Dataset used

- <a href="https://github.com/anshika616/Data-Analysis-Dashboard-2/blob/main/amazon%20sale%20dataset.xlsx">amazon sales data</a>

## Questions (KPIs)

-What is the overall sales performance on Amazon?
-How do sales change by product category and individual products?
-How many orders are cancelled, shipped, or returned?
-Which cities generate the highest sales?
-Which states contribute the most to total sales?
-How do filtered sales compare with total sales?
-Which regions and products are top performers and which need improvement?
